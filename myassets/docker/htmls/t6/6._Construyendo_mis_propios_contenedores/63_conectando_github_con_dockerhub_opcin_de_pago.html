6.3 Conectando Github con DockerHub (Opción de Pago)


Tras la primera edición regional de este curso, la empresa Docker eliminó  la  opción "Automated Buids" del catálogo de operaciones del plan gratuito.

Aunque no vamos a poder realizar este tipo de operaciones, salvo que paguemos,  creo que es una herramienta de gran versatilidad y potencia cuya existencia es interesante conocer. Por ese motivo se ha seguido manteniendo el vídeo en el cuál se "disparaba" un "Automated Build" en DockerHub tras realizar un actualización de un fichero Dockerfile en GitHub.
